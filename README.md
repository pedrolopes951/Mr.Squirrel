# SIDE SCROLLER PROJ 


![Screenshot](Game.png)


## Build

1. Clone repo and create at root dir a build dir `~ mkdir build`
2. Go to build dir and run cmake `~ cd build && cmake .. `
3. Execute make file command at root build dir `~ make`
4. Run executable `./executable/Game`

## Commands 

**W**-Jump 
**S**- Down
**D**- Right
**A**- Left
